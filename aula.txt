git checkout <numero do commit> te leva até o commit com os arquivos que 
havim naquele commit
git checkout -- <arquivo> elimina as mudancas realizadas naquele arquivo
git cheackout HEAD -- <arquivo> elimina as mudancas no estatus de envio	

git revert <commit> - cria um novo commit que desfaz as alteracoes do commit citado

git branch <nome da nova branch>
git checkout branch muda de branch

git tag <nome da tag> cria uma nova tag
git push origin <nome da tag> faz um pull para a tag em questão
